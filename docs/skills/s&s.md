### Параметры и навыки

Но не всё так просто! Сам параметр ограничивает навык сверху:

| Уровень прокачки 	| Требования к параметру 	|
|------------------	|------------------------	|
| Novice           	| 1..2                   	|
| Basic            	| 3..4                   	|
| Advanced         	| 5..6                   	|
| Expert           	| 7..8                   	|
| Master           	| 9..11                  	|
| Grandmaster      	| 12+                    	|
| Legendary        	|                        	|

#### Пример персонажа


|          	|   	|
|----------	|---	|
| ОФП      	| 9 	|
| Ловкость 	| 3 	|
| Разум    	| 5 	|
| Воля     	| 7 	|

С такими параметрами ограничения на начальное значение навыков следующие:

|          	| можно взять только навык, зависящий от: 	|
|----------	|-----------------------------------------	|
| Master   	| ОФП                                     	|
| Expert   	| ОФП, Воля                               	|
| Advanced 	| ОФП, Воля, Разум                        	|
| Basic    	| Навык, зависящий от любого параметра    	|
